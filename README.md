#  M&I Hotel Website

##  Overview
The M&I Hotel website is a professional hotel showcase platform designed to promote luxury accommodation, amenities, and services. It offers users a clean, interactive experience with sections like Home, Rooms, Amenities, Gallery, and Contact.

## Features
- Responsive design using HTML5 & CSS3
- Interactive Gallery with Lightbox and Animations
- Contact Form with HTML5 and JavaScript Validation
- **Interactive Map** showing hotel location
- **Consistent Navigation** across all pages

## Technologies Used
- HTML5 — structure and semantic layout  
- CSS3 — styling, transitions, and animations  
- JavaScript (ES6) — interactivity, form validation, and lightbox functionality  
- Google Maps Embed — location integration  

##  How to Run
* Download/clone the project.

* Open the folder in VS Code.

* Right-click index.html and select "Open with Live Server" (if you have the extension installed).

* Otherwise, double-click index.html to open in your browser.
